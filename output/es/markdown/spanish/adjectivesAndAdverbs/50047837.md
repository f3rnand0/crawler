# Comparación: wie/als

**URL:** https://learngerman.dw.com/es/comparación-wie-als/gr-50047837

**ID:** 50047837

**Category:** Adjetivos y adverbios

## Content

**Expresar igualdad:**  
_so_ \+ adjetivo (forma original/positivo) +_wie_

Si queremos expresar que dos seres vivos o cosas son iguales en cierto
aspecto, usamos _(genau)so_ \+ la forma original del adjetivo + _wie_ :

_Sebastian kocht**so gut wie** Lisa.  
Max arbeitet **genauso viel wie** Tarek._

**Expresar diferencia:**  
Adjetivo (comparativo) + _als_

Si dos seres vivos o cosas se diferencian en algo, usamos el comparativo +
_als_ :

_Tarek kocht**besser als** Sebastian._

_Max putzt**mehr als** Inge._

También podemos usar _nicht_ para expresar diferencia:

_Lisa schläft _**nicht  _so viel wie_** ___ Emma._

# Preposiciones: für

**ID:** 50081892

## Content

Imagen: DW

_Ich habe ein Geschenk für**dich**. Hier, ein Notizbuch. Für **alle neuen
Wörter** auf Deutsch._

La preposición _für_ se utiliza, por ejemplo, para describir el objetivo o el
beneficiario de una acción. Rige acusativo. Tanto los nombres como los
pronombres que vienen después de _für_ tienen que estar en acusativo.

# Temporales: Wie lange...?

**URL:** https://learngerman.dw.com/es/complementos-temporales-wie-lange/gr-50094152

**ID:** 50094152

**Category:** Números

## Content

La duración de una acción responde a la pregunta _wie lange…?_ :

_Wie lange trainierst du?_  
o:  
_Wie lange dauert das Training?_  

A menudo se responde con…

  * la indicación del tiempo transcurrido + **_lang_** :  
  
_Ich trainiere eine halbe Stunde lang_  
  
(La palabra _lang_ puede omitirse: _Ich trainiere eine halbe Stunde._)  

  * el verbo **_dauern_** y la indicación del tiempo transcurrido:  
  
_Das Training**dauert** zwei Stunden._

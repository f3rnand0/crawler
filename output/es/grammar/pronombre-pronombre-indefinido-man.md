# Pronombre indefinido: man

**URL:** https://learngerman.dw.com/es/pronombre-indefinido-man/gr-49826978

**ID:** 49826978

**Category:** Pronombre

## Content

El pronombre indefinido _man_ se usa cuando realizamos declaraciones generales. Expresamos que esa declaración no es válida para una cierta persona, sino para alguien indeterminado, para un grupo o incluso para todos. Aunque _man_ represente a un grupo, se usa en singular. La forma verbal es la misma para _er/sie/es_.

_Nico**isst** gern Gazpacho._

_Er**isst** gern Gazpacho._

_Man**isst** in Spanien gern Gazpacho._

**_man_ y _Mann_**

Presta atención a la diferencia entre el pronombre indefinido _man_ y el nombre _Mann_. El pronombre _man_ , con minúscula y una sola n, se utiliza para la gente en general, de todos los sexos. _Mann_ , en cambio, significa hombre y se refiere a una persona masculina.

Términos gramaticales en alemán das Indefinitpronomen: los pronombres indefinidos se remiten a seres vivos, objetos o situaciones indeterminadas. Se usan cuando no se sabe o no se quiere decir con exactitud quién o cuántas personas hacen algo.

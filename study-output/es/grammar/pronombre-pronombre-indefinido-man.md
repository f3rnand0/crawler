# Pronombre indefinido: man

**Source:** [https://learngerman.dw.com/es/pronombre-indefinido-man/gr-49826978](https://learngerman.dw.com/es/pronombre-indefinido-man/gr-49826978)

## 📖 Overview
El pronombre indefinido man se usa cuando realizamos declaraciones generales. Expresamos que esa declaración no es válida para una cierta persona, sino para alguien indeterminado, para un grupo o incluso para todos. Aunque man represente a un grupo, se usa en singular. La forma verbal es la misma para er/sie/es.

- Términos gramaticales en alemán
- das Indefinitpronomen: los pronombres indefinidos se remiten a seres vivos, objetos o situaciones indeterminadas. Se usan cuando no se sabe o no se quiere decir con exactitud quién o cuántas personas hacen algo

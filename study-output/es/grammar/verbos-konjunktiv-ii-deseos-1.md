# Konjunktiv II: deseos

**Source:** [https://learngerman.dw.com/es/konjunktiv-ii-deseos-1/gr-50115400](https://learngerman.dw.com/es/konjunktiv-ii-deseos-1/gr-50115400)

## 📖 Overview
Cuando hablamos de una situación o circunstancia que no es real, se usa una forma especial del verbo: el Konjunktiv II. Esta forma se usa, por ejemplo, para expresar un deseo.

## 📝 Rules & Explanation
- Cuando hablamos de una situación o circunstancia que no es real, se usa una forma especial del verbo: el Konjunktiv II. Esta forma se usa, por ejemplo, para expresar un deseo
- Términos gramaticales en alemán
- der Konjunktiv II: el Konjunktiv II es una forma verbal con la que se expresa una circunstancia o situación irreal. Se usa, por ejemplo, para hablar sobre deseos o condiciones irreales, pero también para pedir algo con amabilidad o hacer propuestas

# Verbo werden

**Source:** [https://learngerman.dw.com/es/el-verbo-werden/gr-51371625](https://learngerman.dw.com/es/el-verbo-werden/gr-51371625)

## 📖 Overview
Nico will kein Ingenieur werden.
Was willst du werden? Willst du gern Koch werden?

## 📝 Rules & Explanation
- El verbo werden se puede utilizar como verbo auxiliar y como verbo principal. Como auxiliar se usa, por ejemplo, para hacer afirmaciones sobre el futuro
- Con frecuencia, werden se emplea para expresar lo que una persona quiere hacer profesionalmente

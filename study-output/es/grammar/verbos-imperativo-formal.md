# Imperativo: formal

**Source:** [https://learngerman.dw.com/es/imperativo-formal/gr-49995443](https://learngerman.dw.com/es/imperativo-formal/gr-49995443)

## 📖 Overview
Con el imperativo se expresa un pedido, una indicación o una orden. Se utiliza esta forma para, por ejemplo, explicarle a alguien cómo llegar a un determinado sitio. Cuando se trata a una persona de usted, Sie, se utiliza el imperativo formal.

## 📝 Rules & Explanation
- El imperativo formal se forma anteponiendo el verbo conjugado al pronombre personal, colocándolo en la primera posición de la frase
- Términos gramaticales en alemán
- der Imperativ: el imperativo es una forma especial del verbo. Con el imperativo se expresa un pedido, una indicación o una orden

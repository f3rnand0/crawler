# Segundo nominativo

**Source:** [https://learngerman.dw.com/es/segundo-nominativo/gr-50273121](https://learngerman.dw.com/es/segundo-nominativo/gr-50273121)

## 📖 Overview
El sujeto de una frase aparece siempre en nominativo.

## 📝 Rules & Explanation
- El sujeto de una frase aparece siempre en nominativo
- El segundo nominativo describe mejor el sujeto. Solo se puede usar con algunos verbos. Además de sein son p. ej. werden y bleiben. Estos verbos unen el sujeto con un nombre en nominativo

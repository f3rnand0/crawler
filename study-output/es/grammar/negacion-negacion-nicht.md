# Negación: nicht

**Source:** [https://learngerman.dw.com/es/negación-nicht/gr-49896825](https://learngerman.dw.com/es/negación-nicht/gr-49896825)

## 📖 Overview
Para negar un adjetivo se le antepone nicht.

## 🎯 Examples
**German**: nicht
**Context**: Para negar un adjetivo se le antepone

**German**: Das Zimmer ist klein.

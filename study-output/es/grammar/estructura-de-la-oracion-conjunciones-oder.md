# Conjunciones: oder

**Source:** [https://learngerman.dw.com/es/conjunciones-oder/gr-50207775](https://learngerman.dw.com/es/conjunciones-oder/gr-50207775)

## 📖 Overview
Conexión de dos oraciones principales con oder

## 📝 Rules & Explanation
- Si el sujeto y el verbo conjugado son idénticos en ambas frases, en la segunda se pueden omitir

## 🎯 Examples
**German**: oder
**Context**: Conexión de dos oraciones principales con

**German**: oder
**Context**: La conjunción puede unir dos frases principales

**German**: Nehmen Sie das Zimmer für eine Nacht? Bleiben Sie länger?
Nehmen Sie das Zimmer für eine Nacht oder bleiben Sie länger?

**German**: oder
**Context**: Cuando se conectan dos frases con se expresan dos opciones. Estas opciones, por lo general, son excluyen la una a la otra. El orden de las palabras no se altera

**German**: Möchten Sie ein Einzelzimmer? Möchten Sie ein Doppelzimmer?
Möchten Sie ein Einzelzimmer oder ein Doppelzimmer?

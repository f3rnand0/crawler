# Comparación: wie/als

**Source:** [https://learngerman.dw.com/es/comparación-wie-als/gr-50047837](https://learngerman.dw.com/es/comparación-wie-als/gr-50047837)

## 📖 Overview
Expresar igualdad:
so + adjetivo (forma original/positivo) + wie

## 🎯 Examples
**German**: wie
**Context**: Expresar igualdad: so + adjetivo (forma original/positivo) +

**German**: (genau)so
**Context**: Si queremos expresar que dos seres vivos o cosas son iguales en cierto aspecto, usamos + la forma original del adjetivo + wie

**German**: wie
**Context**: Si queremos expresar que dos seres vivos o cosas son iguales en cierto aspecto, usamos (genau)so + la forma original del adjetivo +

**German**: Sebastian kocht so gut wie Lisa.

# Determinante demostrativo: dies-

**Source:** [https://learngerman.dw.com/es/determinante-demostrativo-dies/gr-50081988](https://learngerman.dw.com/es/determinante-demostrativo-dies/gr-50081988)

## 📖 Overview
Con el determinante demostrativo se indica algo específico, se determina con exactitud o se identifica algo dentro de un grupo. A veces se señala con el dedo lo que se quiere expresar con el demostrativo.

## 📝 Rules & Explanation
- Con el determinante demostrativo se indica algo específico, se determina con exactitud o se identifica algo dentro de un grupo. A veces se señala con el dedo lo que se quiere expresar con el demostrativo
